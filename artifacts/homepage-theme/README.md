# Homepage Theme Artifact

This directory contains a sanitized excerpt of the internal Homepage dashboard theme used in the private homelab environment.

Purpose:
- show the information architecture direction
- show the CSS/JS treatment for the operations portal
- keep private topology, live URLs, and service credentials out of the public repo

Included:
- `settings.yaml`
- `widgets.yaml`
- `bookmarks.yaml`
- `services.sanitized.yaml`
- `custom.css`
- `custom.js`

Not included:
- live `services.yaml`
- API keys
- widget credentials
- internal-only runtime files and logs
